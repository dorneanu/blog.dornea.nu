Presentation as Code

19 Jan 2022 | ~ 1 min | tags: [ devops productivity emacs org slides ]

Last year I’ve done some experiments with the Beamer LaTex class which you can use whenever you want to export an ORG document to a Beamer presentation. I’ve done some customizations and I’ve thought I have a quite decent presentation template. Till I’ve found this.

That reveal.js presentation completely my mind and I’ve started digging into asciidoc. And then there is also org-asciidoc which will export your ORG buffer into asciidoc. However there were some issues and quickly after Christmas I decided I’ll go on my own and setup my own revealjs theme.
